# SQL in 5 Days

Learning SQL by working through five case studies on the Chinook database
(a digital music store). Each folder holds a business question, the query,
and the insight. Practiced in VS Code against SQLite.

## Case studies
- Day 1 — Aggregations: where revenue concentrates
- Day 2 — Customer segmentation with CASE
- Day 3 — Joins, hierarchy, and anti-joins
- Day 4 — Window functions: trends and top-N
- Day 5 — CTEs: cohorts and a recursive org chart
